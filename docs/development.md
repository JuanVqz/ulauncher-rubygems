## Development

Clone it.

```bash
gh repo clone JuanVqz/ulauncher-rubygems
```

> If ulauncher is running stop it, right-click the tray ulauncher icon then click exit.

Start it in dev mode with the `Make` command.

```bash
cd ulauncher-rubygems

make dev
```
